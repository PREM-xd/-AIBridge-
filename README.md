AIBridge – Cross-Platform AI Browser Extension

AIBridge is a Chrome browser extension that streamlines AI workflows by providing a unified dashboard for monitoring AI usage across multiple platforms and enabling one-click conversation transfer between AI assistants.

✨ Features

* 📊 Universal AI Usage Dashboard
    * View AI usage information from supported platforms in one place.
    * Track usage windows, reset timers, and available quota (where supported).
* 🔄 One-Click Conversation Transfer
    * Continue conversations across different AI platforms with a single click.
    * Preserve conversation structure and formatting whenever possible.
* ⚡ Cross-Platform Support
    * Designed with a modular provider architecture to support multiple AI assistants, including:
        * ChatGPT
        * Claude
        * Gemini
        * (More providers planned)
* 🧩 Provider Adapter Architecture
    * Each AI platform is integrated through an independent adapter, making the extension easy to maintain and extend.

🛠️ Tech Stack

* React
* TypeScript
* Chrome Extension Manifest V3
* Chrome Extension APIs
* Vite
